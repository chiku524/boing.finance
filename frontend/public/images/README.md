# Boing Network — Outerspace-oceanic assets

Place official Boing brand assets here so the site uses the same theme as the [promotional launch video](https://youtu.be/sSP6wsugauo) and marketing package.

- **boing_background_dark.png** — Hexagonal grid / circuit background (dark mode).  
  The app uses this when no modified version is present.

- **boing_background_dark_modified.png** — Optional. Generated by `scripts/modify_background.py` with different colors and shapes.  
  When present, the app uses this instead of `boing_background_dark.png`.

- **boing_robot_hero.png** — Robot mascot scene (used as source for extraction).

- **hero_elements/** — Extracted 3D objects from the robot hero (run `scripts/extract_robot_hero_elements.py`).  
  Contains `element_0.png`, `element_1.png`, … and `manifest.json`. The app overlays these with 3D motion when this folder and manifest exist.
