# Boing Network — Outerspace-oceanic assets

Place official Boing brand assets here so the site uses the same theme as the [promotional launch video](https://youtu.be/sSP6wsugauo) and marketing package.

- **boing_robot_hero.png** — Source: full Boing hero robot + environment scene.

- **hero_optimized.png** / **hero_thumb.png** — Generated by `scripts/enhance_hero.py`.  
  The app uses the full hero (optimized) in the background layer and the thumb for mascot + footer. Run the script after updating the source PNG.

- **boing_background_dark.png** — Hexagonal grid / circuit background (dark mode).  
  Used when no modified version is present.

- **boing_background_dark_modified.png** — Optional. From `scripts/modify_background.py`.  
  When present, the app uses this instead of `boing_background_dark.png`.

- **hero_elements/** — Optional. From `scripts/extract_robot_hero_elements.py` for alternative layouts.
