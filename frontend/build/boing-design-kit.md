# boing.finance Design Kit & Brand Guidelines

## Brand Overview
boing.finance is a next-generation cross-chain decentralized exchange (DEX) and DeFi infrastructure platform. Our brand represents innovation, reliability, and seamless cross-chain connectivity in the decentralized finance ecosystem.

## Logo Usage

### Primary Logo
- **Format**: SVG (scalable), PNG (transparent background)
- **Usage**: Use on light backgrounds, dark backgrounds, and transparent overlays
- **Minimum Size**: 32px height
- **Clear Space**: Equal to the height of the "B" letter

### Logo Design Elements
- **Central Text**: "BOING" in bold, geometric sans-serif
- **Gradient Fill**: 5-color gradient across all text elements
- **Background**: Circular dark background with subtle gradient overlay
- **Animation**: Subtle animated elements (circles, sparkles, bounce lines)
- **Border**: Circular gradient border with low opacity
- **Effects**: Drop shadows and glow effects for depth

### Logo Variations
1. **Full Logo**: Complete logo with all elements
2. **Icon Only**: Central "B" with network nodes (for small spaces)
3. **Monochrome**: Single color version for single-color applications
4. **Static Version**: Logo without animations (for print/static use)

## Typography

### Primary Font
- **Font Family**: Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto'
- **Usage**: Headings, body text, UI elements
- **Weights**: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)

### Secondary Font
- **Font Family**: 'SF Mono', Monaco, 'Cascadia Code', 'Roboto Mono', monospace
- **Usage**: Code blocks, technical documentation
- **Weights**: 400 (Regular), 500 (Medium)

## Design Elements

### Gradients
- **Primary Gradient**: Multi-stop gradient from #FF6B6B → #4ECDC4 → #45B7D1 → #96CEB4 → #FFEAA7
- **Bounce Gradient**: Linear gradient from #FF6B6B to #FF8E53
- **Usage**: Backgrounds, buttons, highlights, logo elements

### Shadows
- **Light Mode**: rgba(0, 0, 0, 0.05) to rgba(0, 0, 0, 0.1)
- **Dark Mode**: rgba(0, 0, 0, 0.3) to rgba(0, 0, 0, 0.4)
- **Drop Shadow**: rgba(0, 0, 0, 0.3) with 2px offset, 3px blur (from logo)
- **Usage**: Cards, modals, elevated elements, logo effects

### Border Radius
- **Small**: 4px (buttons, inputs)
- **Medium**: 8px (cards, panels)
- **Large**: 16px (containers, major sections)
- **Circular**: 50% (logo background, circular elements)

## Iconography

### Style
- **Type**: Outline icons with 2px stroke width
- **Style**: Modern, minimal, consistent with DeFi aesthetic
- **Library**: Lucide React, Heroicons
- **Animation**: Subtle animations and hover effects (inspired by logo)

### Key Icons
- **Trading**: Arrow up/down, swap arrows
- **Liquidity**: Droplet, pool
- **Bridge**: Link, chain
- **Analytics**: Chart, graph
- **Security**: Shield, lock
- **Bounce**: Animated circles, sparkles (from logo theme)

## UI Components

### Buttons
- **Primary**: Multi-color gradient background (from logo), white text
- **Secondary**: Transparent background, coral red text (#FF6B6B)
- **Success**: Mint green background (#96CEB4), dark text
- **Warning**: Sunshine yellow background (#FFEAA7), dark text
- **Error**: Coral red background (#FF6B6B), white text

### Cards
- **Background**: White (light mode) / Dark gray (#1A1A1A) (dark mode)
- **Border**: Light gray border with subtle shadow
- **Padding**: 24px (large), 16px (medium), 12px (small)
- **Accent**: Subtle gradient border options using logo colors

### Inputs
- **Background**: Light gray (light mode) / Dark gray (#1A1A1A) (dark mode)
- **Border**: Medium gray with focus state in coral red (#FF6B6B)
- **Text**: Dark gray (light mode) / Light gray (dark mode)

## Brand Voice & Tone

### Personality
- **Energetic**: Vibrant and dynamic (reflecting the "bounce" theme)
- **Professional**: Trustworthy and reliable
- **Innovative**: Cutting-edge technology
- **Accessible**: User-friendly and approachable
- **Transparent**: Open and honest communication

### Messaging
- Focus on cross-chain capabilities
- Emphasize security and reliability
- Highlight user experience and ease of use
- Communicate technical innovation clearly
- Convey energy and dynamism (bounce theme)

## Usage Guidelines

### Do's
- Use the full color palette for maximum impact
- Maintain consistent spacing and proportions
- Use gradients for primary elements (especially the 5-color gradient)
- Ensure sufficient contrast for accessibility
- Embrace the energetic, dynamic feel of the brand
- Use subtle animations where appropriate

### Don'ts
- Don't modify the logo proportions or colors
- Don't use colors outside the brand palette
- Don't mix different font families
- Don't use low contrast combinations
- Don't overuse animations (keep them subtle)
- Don't use the logo on busy backgrounds
